## 🏛️ 第一章：起源（1982-1990）

**公司的诞生**

1982 年 12 月，施乐 PARC 的两名科学家约翰·沃诺克和查尔斯·格什克创立了 Adobe 公司。他们以加利福尼亚州 Adobe Creek 小溪命名公司，他们当时就住在那里。他们的目标雄心勃勃：创建一种能够独立于设备描述页面的语言。

**改变印刷业的技术**

PostScript 语言成为印刷业的革命。它以数学方式描述文本和图形，使任何设备都能以相同质量进行打印。这开启了桌面出版系统时代。

```
什么是 PostScript？
==================
PostScript 是一种页面描述语言。它能够：
✓ 将字体描述为数学曲线
✓ 缩放图像而不损失质量
✓ 在任何设备上以相同效果打印
✓ 将文本和图形组合成单一文档
```

**PostScript 对行业的影响**

| 领域 | 变化 |
|------|------|
| 排版 | 数字排字，高质量输出 |
| 设计 | 在计算机上进行排版成为可能 |
| 出版 | 低成本样稿，质量无损 |
| 教育 | 可负担的教材制作 |

**关键事件：与乔布斯的会面**

1983 年，史蒂夫·乔布斯出价 500 万美元收购该公司。沃诺克和格什克拒绝了，但将 20% 的股份出售给苹果公司。这一决定给了 Adobe 财务稳定性。

**后续发展：**

- 1985 年 — Apple LaserWriter（首款 PostScript 打印机）
- 1986 年 — Adobe Illustrator 1.0 发布
- 1987 年 — 字体授权（Times、Helvetica）
- 1989 年 — 诺尔兄弟的第一版 Photoshop

## 🖥️ 第二章：桌面应用时代（1990-2000）

**创意软件的黄金十年**

20 世纪 90 年代是 Adobe 经典产品的繁荣时期。该公司从印刷技术供应商转变为设计师软件创造者。

**Photoshop 的诞生**

1990 年 2 月，Adobe 发布了 Photoshop 1.0。这是一个将永远改变摄影的程序。但很少有人知道它最初只是一个业余爱好项目。

```
Photoshop 创造史
================
1987 年：托马斯·诺尔编写 Display 程序
1988 年：兄弟约翰向乔布斯展示该程序
1989 年：Adobe 获得该程序许可
1990 年：Photoshop 1.0 发布（200 份拷贝）
1994 年：版本 3.0 引入"图层"功能
1998 年：版本 5.0 引入历史记录功能
```

**十年的关键创新**

| 年份 | 创新 | 影响 |
|------|------|------|
| 1990 | Photoshop 1.0 | 首款大众化编辑器 |
| 1993 | PDF 和 Acrobat | 文档标准 |
| 1993 | After Effects | 电脑视觉特效 |
| 1999 | InDesign | 排版新时代 |
| 1999 | Premiere 5.0 | 专业剪辑 |

**对创意产业的影响**

在 20 世纪 90 年代，设计师首次获得了过去需要花费数百万美元才能使用的工具。这导致了：

```
行业变革：
✓ 设计民主化 — 任何人都可以学习
✓ 新一代艺术家的诞生
✓ 网页设计行业的增长
✓ 数字艺术作为一个流派的诞生
```

## 🌐 第三章：数字扩张（2000-2010）

**互联网与多媒体时代**

新千年带来了互联网、视频和数字内容。Adobe 通过扩展其产品线来适应新的格式。

**收购 Macromedia（2005 年）**

这一事件成为转折点。Adobe 以 34 亿美元收购了 Macromedia，获得了：

| 产品 | 用途 | 影响 |
|------|------|------|
| Flash | 网页动画 | 15 年行业标准 |
| Dreamweaver | 网站创建 | 初学者网页设计 |
| Fireworks | 网页图形 | 快速网页设计 |
| Director | 多媒体 | CD 项目 |

**新标准和格式**

```
2000 年代 — 数字化时代：
✓ PDF 成为国际标准
✓ Flash 在 HTML5 出现前是网页动画的基础
✓ 数字视频的出现（Premiere Pro）
✓ 3D 图形处理能力
```

**关键产品发展**

| 年份 | 事件 |
|------|------|
| 2003 | Premiere Pro（面向专业人士的新版本） |
| 2003 | Creative Suite（首个套装） |
| 2005 | 收购 Macromedia |
| 2006 | Dynamic Link（集成） |
| 2007 | Photoshop CS3 发布 |

## ☁️ 第四章：云计算革命（2010-2020）

**向新模式的转型**

2011 年，Adobe 宣布转向订阅模式 — Adobe Creative Cloud。这一决定遭到了批评，但最终被证明是具有远见的。

**Creative Cloud 带来的变化**

```
分发模式：
📦 过去：光盘（每 2 年一次）
☁️ 现在：每月更新（始终保持最新）
📱 过去：仅桌面端
📱 现在：桌面 + 移动应用
💾 过去：本地文件
☁️ 现在：云端库和同步
```

**云时代的新技术**

| 技术 | 年份 | 用途 |
|------|------|------|
| Creative Cloud Libraries | 2013 | 共享资源 |
| Mobile Apps | 2014 | 移动端编辑 |
| Sync | 2015 | 跨平台同步 |
| Adobe XD | 2016 | UX/UI 设计 |
| AI 工具 | 2019 | 智能编辑 |

**Creative Cloud 的优势**

```
对用户而言：
✓ 始终是最新版本
✓ 从任何设备访问
✓ 设置同步
✓ 云存储

对公司而言：
✓ 可预测的收入
✓ 与客户的直接联系
✓ 快速实施创新
✓ 减少盗版
```

## 🧠 第五章：人工智能时代（2020-2026）

**新革命**

2020 年代，Adobe 开始将人工智能整合到所有产品中。这彻底改变了创作方式。

**Adobe Sensei — AI 平台**

Sensei 不是一款独立产品，而是一个嵌入所有 Adobe 软件的平台。它提供：

| 功能 | 描述 |
|------|------|
| 对象识别 | 识别人物、动物、物体 |
| 自动蒙版 | 提取复杂对象 |
| 智能校正 | 建议最佳设置 |
| 内容生成 | 基于提示创建新内容 |

**Adobe Firefly — 生成式 AI（2023 年）**

Firefly 成为首个商业安全的生成式 AI。与竞争对手不同，它仅使用 Adobe Stock 的授权图像进行训练。

```
Firefly 的功能：
✓ 根据文本描述生成图像
✓ 一键更换背景
✓ 生成设计变体
✓ 生成字体和样式
✓ 智能填充空白区域
```

**AI 如何改变设计师的工作**

| 过去 | 现在有 AI |
|------|-----------|
| 在网上搜索图像 | 现场生成 |
| 手动提取对象 | 一键蒙版 |
| 长时间色彩校正 | 自动预设 |
| 复杂修图 | 几秒内神经网络完成 |

## ❓ 常见问题

<details>
<summary><b>Adobe 成立于何时？</b></summary>

Adobe 由约翰·沃诺克和查尔斯·格什克于 1982 年 12 月创立。

</details>

<details>
<summary><b>什么是 PostScript，为什么它很重要？</b></summary>

它是一种页面描述语言，允许在任何设备上打印文本和图形。没有它，就不会有桌面出版系统。

</details>

<details>
<summary><b>Adobe 为什么转向订阅模式？</b></summary>

Creative Cloud 提供持续更新、从任何设备访问和数据同步。这更符合现代需求。

</details>

<details>
<summary><b>什么是 Adobe Firefly？</b></summary>

Firefly 是 Adobe 的生成式 AI 模型，用于根据提示创建图像、文本和效果。它仅使用授权内容进行训练。

</details>

<details>
<summary><b>在 Adobe 中使用生成式 AI 安全吗？</b></summary>

是的，Adobe Firefly 仅使用授权内容，避免版权问题。

</details>

<details>
<summary><b>PDF 支持哪些格式？</b></summary>

PDF 支持文本、矢量图形、光栅图像、3D 模型、交互元素等。

</details>

<summary><b>2026 年 Adobe 有多少订阅用户？</b></summary>

约 3300 万 Creative Cloud 活跃订阅用户。

</details>
